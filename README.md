# 📝 Notes App

A modern full-stack Notes application that allows users to create, edit, delete, and manage their notes easily.

The project focuses on a clean interface, fast performance, and a simple note management experience.

---

## 🚀 Live Demo

You can try the application here:

🔗 [https://your-vercel-link.vercel.app](https://note-web-seven-smoky.vercel.app/)

---

## ✨ Features

* Create new notes
* Edit existing notes
* Delete notes
* Mark notes as completed / not completed
* Automatic date tracking
* Clean and responsive UI

---

## 🛠 Tech Stack

**Frontend**

* React
* Vite
* TypeScript
* TailwindCSS
* React Router
* Framer motion

**Backend**

* Node.js
* Express
* MongoDB
* Rest api
